# Overview

MeshCore is the recommended starting point for most NSW Mesh participants.

Start with the official MeshCore getting started material to understand supported hardware, companion nodes, repeaters, and firmware flashing. Use the MeshCore flasher to install firmware, then come back here before choosing a [frequency profile](./frequency) for NSW Mesh.

## References

- [meshcore.io](https://meshcore.io): MeshCore main website.
- [meshcore.gg](https://meshcore.gg): MeshCore Discord community.
- [docs.meshcore.io](https://docs.meshcore.io): MeshCore documentation reference.
- [MeshCore Flasher](https://meshcore.io/flasher): Flash MeshCore firmware to supported devices.
