# Overview

NSW Mesh started with Meshtastic, but most current local activity has shifted toward MeshCore.

This section needs someone familiar with current Meshtastic practice in NSW to update it with accurate setup notes, channel configuration, device roles, and field operating guidance.

## References

- [meshtastic.org](https://meshtastic.org/): Meshtastic main website.
- [Meshtastic Introduction](https://meshtastic.org/docs/introduction/): Meshtastic documentation introduction.
- [Meshtastic Flasher](https://flasher.meshtastic.org/): Flash Meshtastic firmware to supported devices.

## Help Needed

We need a contributor to review and document:

- Recommended Meshtastic devices for NSW.
- Firmware flashing and upgrade process.
- Channel and region settings.
- Router, repeater, and client role guidance.
- Troubleshooting checklist.
