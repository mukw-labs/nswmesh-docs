# Contributing

Contributions are made through GitHub.

## Editing Pages

1. Open the page you want to improve.
2. Use the "Edit this page" link.
3. Create a branch and submit a pull request.

## Writing Guidelines

- Prefer operationally useful documentation over broad summaries.
- Record assumptions and dates when documenting temporary decisions.
- Keep runbooks step-by-step and test them when possible.
- Link to source discussions or issues for major decisions.
